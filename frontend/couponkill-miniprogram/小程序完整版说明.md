### 🚀 如何使用

#### 1. 开发环境设置
```bash
# 1. 安装微信开发者工具
# 2. 导入项目到微信开发者工具
# 3. 配置你的 AppID（在微信公众平台申请）
# 4. 修改 app.js 中的后端接口地址
```

#### 2. 后端接口配置
编辑 `app.js` 中的 `apiBase`：
```javascript
globalData: {
  apiBase: 'https://your-backend-api.com/api/v1'
}
```

#### 3. 运行项目
- 在微信开发者工具中点击"编译"
- 使用"真机调试"或"模拟器"查看效果

### 🔗 API 接口集成

小程序已配置好与云原生后端的完整集成：

#### 用户认证
- `POST /auth/wechat-login` - 微信登录
- `POST /auth/register` - 用户注册

#### 优惠券相关
- `GET /banners` - 获取轮播图
- `GET /statistics` - 获取统计数据
- `GET /coupons` - 获取优惠券列表
- `GET /coupons/hot` - 获取热门优惠券
- `GET /coupons/seckill` - 获取秒杀优惠券
- `GET /coupons/{id}` - 获取优惠券详情

#### 订单相关
- `POST /orders` - 创建订单
- `POST /orders/seckill` - 秒杀下单
- `GET /orders` - 获取订单列表
- `GET /orders/{id}` - 获取订单详情
- `POST /orders/{id}/cancel` - 取消订单
- `POST /orders/{id}/confirm` - 确认收货

#### 用户相关
- `GET /user/stats` - 获取用户统计
- `GET /user/coupons` - 获取用户优惠券
- `GET /user/orders` - 获取用户订单
