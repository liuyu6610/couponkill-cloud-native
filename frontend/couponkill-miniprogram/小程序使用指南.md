# 🚀 微信小程序使用指南

## 📱 如何运行

### 1. 准备工作
- 安装微信开发者工具
- 注册微信小程序账号并获取 AppID

### 2. 项目配置
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目根目录 `couponkill-miniprogram`
4. 填写你的 AppID

### 3. 配置后端接口
编辑 `app.js` 文件中的 `apiBase`：
```javascript
globalData: {
  apiBase: 'https://your-api-domain.com/api/v1'
}
```

### 4. 运行项目
- 点击"编译"按钮
- 使用"真机调试"或"模拟器"查看效果

## 🔗 API 接口规范

小程序遵循 RESTful API 设计：

- `GET /banners` - 获取轮播图
- `GET /coupons` - 获取优惠券列表
- `GET /coupons/seckill` - 获取秒杀优惠券
- `POST /orders` - 创建订单
- `GET /orders` - 获取订单列表
- `POST /auth/wechat-login` - 微信登录

## 💡 使用技巧

1. **登录授权**：首次使用需要微信授权登录
2. **网络权限**：需要在小程序管理后台配置合法域名
3. **支付功能**：集成微信支付，可直接购买
4. **分享功能**：支持分享优惠券给好友

## 🚨 注意事项

1. **域名配置**：后端接口域名需要在小程序后台配置
2. **HTTPS 要求**：生产环境必须使用 HTTPS 协议
3. **支付签名**：支付功能需要正确的商户配置
4. **版本更新**：新版本发布需要重新审核
